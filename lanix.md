# Lanix Neuron V — Hardware

| Componente | Detalle |
|---|---|
| **Equipo** | Lanix Neuron V (Clevo, ~2012) |
| **CPU** | Intel Core i7-3520M @ 2.90GHz (Ivy Bridge, 2C/4T) |
| **RAM** | 16 GB |
| **Disco** | 446 GB (LVM cifrado, 8% usado) |
| **GPU** | Intel HD Graphics (3rd Gen, i915) |
| **Red** | Realtek RTL8111 Gigabit Ethernet |
| **WiFi** | Realtek RTL8723AE PCIe |
| **Audio** | Intel HDA (7 Series) |
| **Dock** | Dell D3100 USB3.0 (DisplayLink) |
| **Kernel** | 7.0.9-arch2-1 (Arch Linux) |
| **Puertos video** | HDMI, VGA (nativos, card1) |

## Notas
- Sin Thunderbolt — docks TB3/TB4 no compatibles
- Video del dock D3100 va por DisplayLink (evdi), no soportado en Hyprland/wlroots
- Monitor externo funciona conectado directo por HDMI al laptop
